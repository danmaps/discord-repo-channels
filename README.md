# Discord Repo Channels

Create structured, repeatable conversations around repos — with optional automation to create a per‑repo Discord channel.

## What’s inside

- **CONVERSATION.md** — drop‑in templates for issues, PRs, file discussions, and action logs
- **Automation sketch** — minimal webhook pattern to auto‑create `#proj-<repo>` channels

## Quick start

1. Copy `CONVERSATION.md` into any repo
2. Fill in the Repo Metadata section
3. Use the templates to keep conversations clean
4. (Optional) wire up a webhook to create channels automatically

## Why

Most teams lack a consistent way to capture decisions, open questions, and next actions. This gives you a light‑touch structure that works in GitHub, Discord, or both.
